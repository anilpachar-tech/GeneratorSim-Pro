# ⚡ GeneratorSim Pro

GeneratorSim Pro is an industrial-style generator simulation dashboard developed using Python and Streamlit. It is designed to simulate and analyze the performance of electrical generators under different operating conditions, faults, and resistance variations.

---

## 🚀 Features

- 🔌 Synchronous Generator Simulation  
- ⚙️ DC Generator Simulation  
- ⚠️ Fault Simulation (Voltage Drop, Overload, Overspeed, etc.)  
- 🧮 Resistance Variation (Field & Armature Effects)  
- 📊 Real-time Performance Metrics  
- 🌀 Animated Rotor Visualization  
- 📈 Advanced Performance Graphs  
- 📋 Detailed Output Table  
- 📁 Export Results as CSV  
- ⚖️ Comparison Mode (Generator A vs Generator B)  
- 🟢 Health Status Indicator (Excellent / Moderate / Critical)
